Project Overview

SafePal-AI is a real-time scam detection mobile application that monitors ongoing phone calls, transcribes conversations, performs speaker diarization, analyzes call intent using a Large Language Model (LLM), and instantly alerts users if scam activity is detected â€” while the call is still in progress.

This is not a spam-blocking app.
It is a conversation-understanding AI safety system.
